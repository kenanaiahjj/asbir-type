# Asbir Mono Review 1.0.0

This is the complete **review** package for Asbir Mono. It is not an approved production release. The core family includes nine static weights in TTF and CFF OTF plus a `wght` 100–900 variable TTF.

The coding ligatures `->`, `=>`, `!=`, `==`, `===`, `<=`, and `>=` are enabled by default through `liga`; disable standard ligatures to show literal ASCII operators. The core font also includes `⇐`, `⇒`, and `⇔`.

`fonts/terminal/AsbirMono-NerdFont-Regular.ttf` is a separately named, fixed-cell Regular terminal font with Nerd Font / Powerline symbols. Use it for terminals and developer tooling, not as a substitute for the core variable family.
