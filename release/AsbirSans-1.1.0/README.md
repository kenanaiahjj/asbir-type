# Asbir Sans 1.1.0

Approved production release. Includes nine static Roman and true italic weights in TTF and CFF OTF, plus separate Roman and italic variable TTFs. The Sans variable files expose `wght` (100–900) and `opsz` (14–32) axes.

Install the static fonts for fixed-weight environments, or use `Variable/AsbirSans-Variable.ttf` or `Italic/AsbirSans-Italic-Variable.ttf` in variable-font-capable software. The `web/AsbirSans/` folder contains the WOFF2 and CSS loading kit.
