# Asbir Sans 1.0.0

Approved production release. Includes nine static weights in TTF and CFF OTF, plus a variable TTF with `wght` (100–900) and `opsz` (14–32) axes.

Install the static fonts for fixed-weight environments, or use `Variable/AsbirSans-Variable.ttf` in variable-font-capable software.
