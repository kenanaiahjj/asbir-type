# Asbir family directions

## Asbir Sans

High-x geometric grotesk for interface and everyday reading. The shared family anchor: large lowercase, open counters, firm terminals, neutral but authored texture.

## Asbir Serif

Display-led companion with contemporary old-style contrast. It takes Instrument Serif only as a reference for condensed editorial tension; it keeps Asbir's higher lowercase, measured spacing, and more controlled geometry. Start glyphs: `H O n o a e s R g &`.

## Asbir Mono

Practical UI/code companion rather than a nostalgic terminal font. Keep a high
x-height, steady widths, large punctuation, slashed `0`, a two-storey `a`,
and explicit differentiation for `I l 1`, `O 0`, `{ }`, and `()`. The core
coding operators `->`, `=>`, `!=`, `==`, `===`, `<=`, and `>=` use default
ligatures while preserving their literal ASCII form when `liga` is disabled.
Include the Unicode technical arrows `⇐ ⇒ ⇔`. A separately named Nerd Font /
Powerline terminal TTF carries icon symbols; it is not part of the core family.
Start glyphs: `H O n o a e 0 1 I l { } @ %`.
